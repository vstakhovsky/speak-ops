# SpeakOps Assets

This directory contains visual assets for the SpeakOps repository.

## Files

- `speakops-architecture.svg` — Repository architecture diagram (terminal/editor-inspired)
- `speakops-core-loop.svg` — Core activation loop visualization
- `README.md` — Assets overview

## Color Palette

All SVGs use this professional palette for consistency:

```
Background:  #11181C  (dark blue-gray)
Panel:        #172126  (slightly lighter)
Text:         #D8DEE3  (light gray-blue)
Muted:        #8A969E  (subtle gray-blue)
Blue-gray:    #6F95A5  (calm blue)
Gold:         #B7A27A  (accent gold)
Clay:         #B97868  (warm brown)
Green:        #8FAE9A  (subtle green)
Purple:       #B78BEA  (accent purple)
```

## Design Principles

- **Dark theme optimized** — All SVGs use dark backgrounds
- **Terminal-inspired** — Monospaced typography, editor-style layout
- **Label-heavy** — Clear labeling for each component
- **GitHub-compatible** - Works in both light and dark themes
- **Professional** — Clean, minimal, technical aesthetic

## Usage

Include in README.md:

```markdown
<p align="center">
  <img src="./assets/speakops-architecture.svg" alt="SpeakOps architecture" width="900">
</p>
```
